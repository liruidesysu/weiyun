
2.资源概况
^^^^^^^^^^^^^^^^
  **<资源概况>** 页面显示的内容为当前一周的课程表以及本周桌面调度数量的百分比。
  
  管理员可以从课程表中得知当前或者某一时间段内开启多少门课程。
  
  教师可以在课程表中查看并核实自己所属课程创建时间正确。
  
  如当前时间段开启课程后：
  
  **当0%<课程桌面数量<30%时，课程表的表格为绿色；**
  
  **当30%<课程桌面数量<60%时，课程表的表格为橙色；**
  
  **当60%<课程桌面数量<100%时，课程表的表格为红色。**
