
14.系统日志
^^^^^^^^^^^^^^

  系统日志中，可以查看到用户操作日志和系统运行日志。管理员可以通过日志监控系统运行情况。

14.1.用户操作日志 
----------------------

  **用户操作日志** 记录了用户登录、登出、管理用户、管理镜像、管理桌面等等操作，管理员可以在这里查看何时进行了何种操作，以了解系统情况以及对系统进行合理管理的目的。
  
  在 **<用户操作日志>** 界面，可以对日志进行搜索，在右上角搜索框输入要搜索的关键词，并且在日期选择框输入查询的时间段  **（不选择日期则默认查询当天的记录）** ，点击 **【查询】** 按钮会得到想查看的结果。
