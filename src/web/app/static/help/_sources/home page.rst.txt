
1.我的主页
^^^^^^^^^^^^^^^
  
  **<我的主页>** 主要分为三部分，
  
  第一部分为资源在线使用数据，包括 **在线主机数量、今日上课数量、在线用户数量** 以及 **在线桌面数量**。
  
  第二部分为授权数量与当前系统使用数量的饼状图对比，包括：**CPU使用情况、内存使用情况、磁盘使用情况** 以及 **镜像缓存使用情况**。
  
  第三部分为曲线图，可实时记录CPU、内存、磁盘数据的变化。
  
  .. image:: /images/zhuye_1.png
     :alt: 我的主页
     :align: center	
	 
  图1-1 我的主页
  
  点击 **在线主机数量**，页面将跳转到<监控管理>页面；
  
  点击 **今日上课数量**，页面将跳转到<课程管理>页面；
  
  点击 **在线用户数量**，页面将跳转到<用户管理>页面；
  
  点击 **在线桌面数量**，页面将跳转到<桌面管理>页面。
